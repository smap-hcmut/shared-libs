# Usage Examples

This directory contains comprehensive usage examples for all shared library packages.

## Structure

- `go/` - Go service integration examples
- `python/` - Python service integration examples
- `cross-language/` - Cross-language trace propagation examples
- `migration/` - Step-by-step migration examples

## Examples Available

### Go Examples
- Basic trace context usage
- HTTP client with trace propagation
- Kafka producer/consumer with trace headers
- Database operations with trace logging
- Complete service integration

### Python Examples  
- FastAPI middleware integration
- Async HTTP client usage
- Kafka message processing
- Database operations with trace context
- Complete service integration

### Cross-Language Examples
- Go service → Python service trace flow
- HTTP → Kafka → Database trace propagation
- Error handling and recovery scenarios
- Performance optimization patterns

## Quick Start

Each example includes:
- Complete working code
- Setup instructions
- Expected output
- Common troubleshooting tips
